國小班級座位表管理工具 - 使用說明書

老師您好！為了讓班級經營與課堂互動更加輕鬆高效，我們開發了這款多功能的座位表管理工具。它不僅僅是一個座位表，更是一個整合了分組、計分、排行與互動回饋的教學好幫手。

✨ 核心功能總覽

  • 📊 分數即時排行：個人與分組總分排行榜，動態更新，激發學習動機。
  • 🖱️ 直覺拖曳換位：用滑鼠直接拖曳學生即可交換座位，並能自動更換組別。
  • 🎨 智慧分組管理：提供多種分組方式（隨機、同組同行），並以顏色區分。
  • 🔊 互動即時回饋：點擊加減分時，有數字動畫與提示音效，操作確認更清晰。
  • 💾 檔案與瀏覽器存取：可透過 Excel 檔案或瀏覽器內建功能，輕鬆存取、備份與還原資料。
  • 📄 專業列印排版：專為列印設計的版面，可將計分板與座位表清晰地分頁列印。
  • 🛡️ 防呆存檔提醒：在您離開網頁前，若有未儲存的修改，系統會貼心提醒。

🚀 第一步：準備您的班級名單 (Excel 檔案)

本工具現在透過匯入 Excel (.xlsx, .xls) 檔案來建立班級名單，可完全避免中文亂碼問題。

  1. 建立檔案：用 Excel 或 Google Sheets 建立一個試算表。
  2. 設定欄位標題：在檔案的第一個工作表中，其第一列必須是以下標題（順序不拘，但名稱需完全相符）：
    標題說明範例
    科目目前的課程科目國語
    班級您的班級名稱三年二班
    座號學生的座號1
    姓名學生的姓名王小明
    分組學生所屬組別（可先留白）第1組
    組長是否為組長（填 是 或 否）否
    分數學生的起始分數（可填 0）0

💾 第二步：資料管理 (匯入、匯出、存取)

  • 匯入 Excel 名單：從您的電腦選擇準備好的 Excel 檔，即可載入單一班級的座位表。程式會自動計算並調整所需列數。
  • 匯出 Excel：將目前畫面上的座位表（包含所有學生、分數、組別與空位）完整匯出成一個 Excel 檔。適合用來備份單一課程的狀態。
  • 儲存到瀏覽器：將目前畫面上的狀態，用一個自訂名稱（例如：三年二班-國語）儲存在您這台電腦的瀏覽器中。
  • 從瀏覽器讀取：彈出列表，顯示所有已儲存的紀錄，您可以快速讀取或刪除。
  • 匯出所有存檔：將您存在瀏覽器中的**所有紀錄**，一次全部打包成一個 .json 備份檔，方便您轉移到另一台電腦。
  • 匯入所有存檔：選擇 .json 備份檔，可將所有紀錄一次還原到新的電腦瀏覽器中（同名紀錄會被覆蓋）。

🎨 第三步：座位與分組管理

  • 拖曳座位：直接用滑鼠拖曳學生即可交換位置。學生會自動加入新位置所在行的組別。
  • 指定組長：點擊座位上的 星號(⭐) 即可指定或取消該組的組長。
  • [同組坐同行]：點擊後，系統會將所有學生依照組別編號重新排列，並用空位補滿該行。
  • [隨機分組]：點擊後，系統會根據您設定的「每組人數(行)」，將所有學生隨機打亂並分配到新的組別中。
  • [重置座位表]：當您手動修改「組數」或「人數」後，點此按鈕套用變更。若您縮小版面，多出的學生會被保留並移至下方「未分組」區域。

💡 溫馨提示：建議的工作流程是：建立好 Excel 名單 → 匯入 → 使用分組功能 → 課堂中計分 → 使用「儲存到瀏覽器」快速存檔。若要換電腦，再使用「匯出/匯入所有存檔」功能。
